CSS Specificity

La spécificité est le moyen par lequel les navigateurs décident quelles valeurs de propriété CSS d'un élément sont les plus pertinentes et seront appliquées.
La spécificité est basée sur les règles, ciblant un élément, composées par différents types de sélecteurs CSS.

Considérez la spécificité comme un score/rang qui détermine quelle déclaration de style est finalement appliquée à un élément.

Lorsqu'une règle !important est utilisée dans une déclaration de style, cette déclaration est prioritaire par rapport à toute autre déclaration.

!important > style="" > #id > .class > html
