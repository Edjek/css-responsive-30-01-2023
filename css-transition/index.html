Les transitions CSS

Les transitions CSS permettent de contrôler la vitesse d'animation lorsque les propriétés CSS sont modifiées.
Plutôt que le changement soit immédiat, on peut l'étaler sur une certaine période.
Ainsi, si on souhaite passer un élément de blanc à noir, on pourra utiliser les transitions CSS afin que cette modification soit effectuée progressivement, selon une courbe d'accélération donnée.
